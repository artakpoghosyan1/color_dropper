# Color Dropper

## Tech stack

- **Pure JavaScript**.
- **TypeScript**
- **Webpack**
- **Babel**

## Getting Started

### Prerequisites

Ensure you have Node.js and npm installed on your machine.

Clone the repository:

```bash
git clone https://github.com/artakpoghosyan1/color_dropper.git
cd color_dropper
```

### Run the project

```bash
npm i
npm run start
```
